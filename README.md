# เที่ยงทำ ใบเสนอราคา (Thiengtham Quotation App)

## 📋 คำอธิบาย
แอปพลิเคชันจัดการใบเสนอราคาสำหรับช่างพรชัย ที่ช่วยให้สร้างและจัดเก็บใบเสนอราคาได้ง่ายขึ้น

## ✨ ฟีเจอร์
- ✅ สร้างใบเสนอราคาใหม่
- ✅ คำนวณราคารวม VAT อัตโนมัติ
- ✅ แปลงตัวเลขเป็นภาษาไทย (Baht Text)
- ✅ บันทึกข้อมูลใน Local Storage
- ✅ ดูประวัติใบเสนอราคา
- ✅ Dark theme ที่สวยงาม
- ✅ Mobile-friendly design

## 🛠️ ใช้เทคโนโลยี
- **React** - UI Framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **Lucide React** - Icons

## 📁 โครงสร้าง
```
thiengtham-quotation-app/
├── index.html           # เวอร์ชั่น HTML ธรรมชาติ
├── src/
│   └── App.tsx         # React Component หลัก
├── package.json        # Dependencies
└── README.md          # ไฟล์นี้
```

## 🚀 เริ่มต้นใช้งาน

### ด้วย HTML
```bash
# เปิดไฟล์ index.html ในเบราว์เซอร์
open index.html
```

### ด้วย React
```bash
# ติดตั้ง dependencies
npm install

# รัน development server
npm run dev
```
